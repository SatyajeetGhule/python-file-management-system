# Python File Management System
A simple **Command Line Interface (CLI) File Management System** built using Python.
This project allows users to perform basic file operations such as creating, reading, editing, deleting, and viewing files in a directory.

---

## 📌 Features

* Create a new file
* View all files in a directory
* Read file content
* Edit / Append data to a file
* Delete a file
* Command-line menu interface
* Exception handling for safe file operations

---

## 🛠 Technologies Used

* Python 3
* OS Module
* File Handling

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/python-file-management-system.git
```

2. Go to the project directory

```
cd python-file-management-system
```

3. Run the program

```
python3 app.py
```

## ⚙️ Functions Used

* `create_file()` – Create a new file
* `view_all_files()` – Display all files in a directory
* `read_file()` – Read and display file content
* `edit_file()` – Append data to a file
* `delete_file()` – Remove a file
